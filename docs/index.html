Install

Get started in under a minute

No Python needed — Patch is bundled inside a notarized app, so it opens without any Gatekeeper warning.

🔒 Patch only ever reads ~/.claude usage logs and the OAuth token in your Keychain — it never touches Photos, Downloads, or Documents.
Calibration

Calibrated to your real limits

Token limits are private to Anthropic, so Patch can't know them outright. Type the % shown in Claude's Settings → Usage, and Patch back-solves the rest: limit = current usage ÷ %.
